## App Review Notes

Annai Travel Planner is a travel planning app with one optional monthly subscription called Annai Pro.

Core free features:
- trip creation and editing
- itinerary planning
- packing lists
- budget tracking
- document vault

Annai Pro features:
- Ask Annai travel assistant
- AI trip plan generation
- AI destination info tools
- premium map/location features

### Where to find the subscription

1. Sign in or create an account
2. Open Pricing or Account
3. Tap "Start Annai Pro"

### Important reviewer notes

- Account deletion is available inside the Account page
- Public support page: `https://annaiapp.com/support/`
- Public privacy policy: `https://annaiapp.com/privacy-policy/`
- Public terms: `https://annaiapp.com/terms-of-service/`
- Public account deletion page: `https://annaiapp.com/account-deletion/`
